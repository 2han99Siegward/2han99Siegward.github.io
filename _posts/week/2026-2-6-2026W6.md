### AI 开始「雇佣人类」：RentAHuman.ai 爆火，数百人排队上架
#ai  
昨天，一个名为「租个人吧」（RentAHuman.ai）的新 platform引发广泛关注。

该平台以「出租人类」为核心 concept(概念)，目标用户并非传统意义上的消费者，而是运行在服务器中的AI智能体。 平台上线后短时间内吸引数百名注册者，访问量一度导致服务器 crash(宕机)。

RentAHuman.ai 的开发者表示，平台旨在为 AI 提供触达物理世界的「肉身层」（The Physical Layer）。

随着 OpenClaw 等开源 AI Agent 在今年爆火，AI 已能自主执行写代码、浏览网页、谈判甚至 transaction(交易)等复杂任务，但依然无法完成现实世界中的物理操作，如取干洗衣物、查看咖啡厅是否拥挤等。

在该平台上，AI 可以通过 MCP（Model Context Protocol）发起任务请求，并以稳定货币支付每小时 50 至 200 美元不等的 fee，由人类执行任务。

### OpenAI 推出 macOS 版 Codex 应用：整合智能体化开发逻辑，剑指 Claude Code 用户
#ai  
当下的主流趋势是智能体化软件开发，即人工智能智能体可独立完成编码任务的系统，Claude Code 与 Cowork 这类应用便是该模式的典型代表。与此同时，OpenAI 也在逐步完善其 Codex 工具，该工具于去年 4 月以命令行工具的形式推出，一个月后便拓展出网页端界面。

如今 OpenAI 正式迈出追赶的关键一步。当地时间本周一，该公司推出了适配 macOS 系统的全新 Codex 应用，整合了过去一年间广泛流行的各类智能体化开发逻辑。这款新应用支持多智能体并行作业，可融合不同智能体的能力，以及当前最前沿的工作流程。此次发布距离 OpenAI 推出其最强编码大模型 GPT-5.2-Codex，尚不足两个月，公司希望凭借该模型吸引 Claude Code 的用户。
